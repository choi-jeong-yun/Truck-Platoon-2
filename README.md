# Truck-Platoon-2
SEA:ME@KOREA TRUCK PLATOONING 2기 코드 정리

truck_control 폴더의 python 코드를 정리했습니다.

+ truck_platooning 폴더의 C++ 코드도 올렸습니다. (미완성, 수정 필요)

현재 구현 기능: 트럭 군집주행, 차선 변경 후 합류 시나리오


  - t: 전체 재배치(왼쪽) — 선두 차량이 좌측 차선으로 빠진 뒤 순서 재배치
  - y: 전체 재배치(오른쪽)
  - j: 후행차량(현재 2번 위치 차량)을 선두로 올림 (왼쪽 차선만)
  - k: 후행차량(현재 3번 위치 차량)을 선두로 올림 (왼쪽 차선만)
  - q / e: Truck 0 왼쪽 / 오른쪽 개별 차선 변경
  - a / d: Truck 1 왼쪽 / 오른쪽 개별 차선 변경
  - z / c: Truck 2 왼쪽 / 오른쪽 개별 차선 변경
  - ESC: 종료
